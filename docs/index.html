AI Pricing Engine

Interactive pricing intelligence dashboard for business strategy, demand elasticity, and customer sentiment.

Project Overview

This project combines transaction data, sentiment analysis, and economics to recommend pricing actions. It is built as a Streamlit dashboard and documented for GitHub portfolio presentation.

Live App

Run the dashboard locally with streamlit run app.py.
