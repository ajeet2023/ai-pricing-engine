# ml_pricing.py

## Purpose

This file contains the economic and statistical core of the project.

## Main responsibilities

- Estimates demand elasticity with a log-log OLS model
- Computes a profit-maximizing price from the elasticity estimate
- Builds a profit curve for scenario analysis
- Converts the model output into simple business recommendations

## Why it matters

This is the differentiator layer of the project. It shows the business logic behind the dashboard and makes the work look much more analytical than a standard visualization tool.
