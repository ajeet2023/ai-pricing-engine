# Dashboard Guide

## What the dashboard shows

The Streamlit dashboard gives a business-friendly view of pricing decisions. It highlights:

- transaction volume and average value
- demand elasticity estimates
- an optimal price recommendation
- a profit curve for scenario analysis
- a sentiment-based business insight section

## How to use it

1. Run the app with `streamlit run app.py`.
2. Adjust the cost and elasticity sliders.
3. Review the recommendation and profit curve.
4. Use the output to support price decisions or portfolio storytelling.

## Business value

This dashboard translates analytics into a simple story:

- If demand is elastic, lowering price may increase volume.
- If demand is inelastic, a modest price increase may be acceptable.
- If sentiment is weak, customers may react negatively to price changes.
